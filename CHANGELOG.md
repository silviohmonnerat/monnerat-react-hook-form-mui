## [0.1.0] - 2022-03-12

This project simplifies the use of `react-hook-form` and `Material-UI`. It provides opinionated use cases with following components:

### Added

- FormContainer
- TextFieldElement
- SelectElement
- MultiSelectElement
- RadioButtonGroup
- CheckboxButtonGroup
- CheckboxElement
- SwitchElement
- PasswordElement
- DatePickerElement

`The components are written in TypeScript and attempt an easy integration.`
